## 👨‍💻 Sobre Mim
Atualmente, foco na intersecção entre o desenvolvimento de aplicações robustas e a inteligência de dados. Sou especialista em **Engenharia de Dados com MS SQL Server**, onde projeto pipelines de ETL/ELT críticos, enquanto construo ecossistemas modernos utilizando **Node.js, C# e React**.
